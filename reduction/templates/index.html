🎯 Absolute calibration constant K
cm · (monitor units)⁻¹
Applied as: dΣ/dΩ (cm⁻¹) = K × counts / (Bstop × dcm)
How to determine K: measure a primary standard under the same conditions, then compute K = Iref(q) / [counts(q) / (Bstop × dcm)] at a q-range where the reference value is known.
 • Water — use the flat plateau at high-q (e.g. ~1.65 × 10⁻² cm⁻¹ at 25 °C).
 • Glassy carbon (GC) — use the absolute dΣ/dΩ curve supplied by your beamline (NIST SRM 3600 recommended).
Leave K = 1.0 for semi-absolute output (relative cm⁻¹ units, proportional to dΣ/dΩ but not calibrated).
⚠ Warning: I₀ + Bstop together is non-physical — Bstop already contains the I₀ signal (Bstop ∝ Φ₀·T), so selecting both double-counts incident flux. Use one or the other.
What each mode computes
None — raw integrated counts, no correction
I(q) = counts(q)
Output is in detector counts. Use for diagnostics or when applying all corrections externally (e.g. in a downstream analysis script).
Bstop — divide by transmitted-beam monitor
I(q) = counts(q) / Bstop
Bstop ∝ Φ₀·T (incident flux × sample transmission), so this simultaneously corrects for beam intensity variations and sample absorption in one step. Recommended default for SSRL beamlines. Physically: I(q) ∝ dΣ/dΩ · (constant).
I₀ — divide by incident-beam monitor
I(q) = counts(q) / I₀
Corrects for incident flux Φ₀ only; does not account for sample transmission. The result still carries the absorption factor T. Useful when Bstop is unavailable or unreliable. Do not combine with Bstop (double-counts Φ₀) or with Absolute (Absolute already contains Φ₀ via B_stop ∝ Φ₀·T).
Absolute — semi-absolute intensity (cm⁻¹)
I(q) = counts(q) / (Bstop · dcm) ∝ dΣ/dΩ [cm⁻¹]
Since Bstop ∝ Φ₀·T and T = exp(−μd), dividing by Bstop·d gives intensity proportional to the macroscopic differential scattering cross-section dΣ/dΩ in cm⁻¹. A calibration constant K (from a primary standard such as water or glassy carbon) is still required for true absolute units:
dΣ/dΩ (cm⁻¹) = K · counts(q) / (Bstop · dcm)
Mutually exclusive with both Bstop and I₀ — Absolute already incorporates both (B_stop ∝ Φ₀·T subsumes Bstop and I₀ in one number). Selecting Absolute automatically deselects the other two.
Click pills to toggle. Active pills are highlighted. The active terms are multiplied together as PyFAI's normalization_factor (PyFAI divides all pixel counts by this scalar before azimuthal averaging).
